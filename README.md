# E-LEARNING-PORTAL

Full-stack e-learning portal project with a React frontend and Node.js/Express backend.

## Project Structure

- `frontend/` - React application
- `backend/` - Express API and MongoDB backend

## Tech Stack

- React
- Node.js
- Express
- MongoDB

## Run Locally

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm start
```

## Notes

- Keep your environment variables in local `.env` files.
- `node_modules`, logs, uploads, and build files are ignored from Git.
